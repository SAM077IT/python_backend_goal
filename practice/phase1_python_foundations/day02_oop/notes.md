## Day 2 — OOP: Classes, Inheritance, Dunder Methods

### Key takeaways
- `__init__` is called automatically when you create an object
- `super()` lets a child class call the parent's method
- Dunder methods like `__str__`, `__repr__`, `__len__` let you
  customize built-in behavior
