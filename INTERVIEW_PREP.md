# 🎤 Interview Prep — Question Bank

> Study these topics from Week 10 onwards. But keep them in mind from Day 1.

---

## 🐍 Python Core Questions

### Must Know
1. What is the GIL? How does it affect multi-threading in Python?
2. How does Python manage memory? What is reference counting?
3. What are decorators? Write one from scratch.
4. What is the difference between `@staticmethod` and `@classmethod`?
5. What are generators? When would you use `yield` over `return`?
6. Explain list comprehensions vs generator expressions.
7. What is `*args` and `**kwargs`?
8. Difference between `deepcopy` and `shallow copy`?
9. What are Python's mutable vs immutable types?
10. What is `__slots__`? When would you use it?

### OOP Questions
11. Explain the four pillars of OOP with Python examples.
12. What is multiple inheritance? How does Python's MRO work?
13. What are dunder/magic methods? Give 5 examples.
14. What is duck typing?
15. Explain SOLID principles with a Python example.

## 🌐 Django / DRF Questions

1. Explain Django's MVT architecture.
2. What is the difference between `ForeignKey`, `OneToOneField`, and `ManyToManyField`?
3. How does Django's ORM work? What is `select_related` vs `prefetch_related`?
4. What is middleware in Django? Give examples of custom middleware.
5. Explain Django signals. When would you use them?
6. What is the N+1 query problem? How do you fix it?
7. How does DRF serializer validation work?
8. What is the difference between `ModelViewSet` and `APIView` in DRF?
9. How do you implement pagination in DRF?
10. How does Django's authentication system work?

## 🔢 DSA Topics & Priority

### High Priority (appears in most Indian company interviews)
- Arrays & Strings
- Two Pointers
- HashMap / HashSet
- Sliding Window
- Stack & Queue
- Binary Search

### Medium Priority
- Linked Lists
- Trees (BFS, DFS)
- Recursion
- Sorting (merge sort, quick sort)

### Lower Priority for 3-Month Timeline
- Dynamic Programming (know Fibonacci + knapsack basics)
- Graphs
- Heaps / Priority Queues

## 🎯 Interview Rounds — What to Expect

### Typical Indian Company Process
1. **Online Assessment** — Aptitude + 2 coding problems (DSA, easy/medium)
2. **Technical Round 1** — DSA problems + Python basics
3. **Technical Round 2** — Project discussion + system design basics
4. **HR Round** — Behavioral + salary negotiation

### Tips
- Always talk through your thought process aloud
- If stuck, ask for a hint — it's okay
- Know your projects inside-out (they WILL ask)
- Have your GitHub open on your second screen
